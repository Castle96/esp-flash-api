Features

🎤 Wake→ 📡 ESP32→ 📝 STT→ 🧠 LLM→ 🔊 TTS→ 🔁 Reply
🎤

Voice Control

Push-to-talk from ESP32-C3 with INMP441 mic and I2S speaker. Full STT → LLM → TTS pipeline.

⚡

OTA Flashing

LLM generates MicroPython firmware. Human-in-the-loop approval. Wireless OTA push to fleet.

📊

Dashboard

Catppuccin Mocha SPA — real-time pipeline viz, device fleet, flash queue, conversation log, weather.

🔧

Tool Calling

LLM can list devices, flash firmware, set reminders, and check system status via structured tools.

🔄

CI/CD Integration

Gitea webhooks auto-create flash jobs on firmware push. Build status on device cards.

🔐

No Cloud

faster-whisper, Kokoro, Ollama, openWakeWord — all on your hardware. No API keys, no subscriptions.

Quick Start

git clone http://100.120.207.67:30529/Castle96/esp-flash-api.git
cd esp-flash-api/diva

# Configure
cp backend/.env.example backend/.env
# Edit backend/.env with your machine IPs

# Run tests (no hardware)
make test

# Deploy
make up-all
make smoke
